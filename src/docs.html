Installation npm pnpm yarn bun Run the following command using npm: 
npm i wireframes
 Run the following command using pnpm: 
pnpm i wireframes
 Run the following command using yarn: 
yarn add wireframes
 Run the following command using bun: 
bun add wireframes
 Setup After installing, you need to set up at the root of your application. This can be either in your index.jsx, index.tsx or App.jsx depending on the framework you use. 
...

import { applyPolyfills, defineCustomElements } from 'wireframes/loader';
import 'wireframes/dist/wireframes/wireframes.css';

applyPolyfills().then(() => {
  defineCustomElements();
});

...
CDN Alternatively, you can use the CDN to load the components in your project. 
<script type="module" src='https://www.unpkg.com/wireframes@latest/dist/wireframes/wireframes.esm.js'></script>

<link rel="stylesheet" href="https://www.unpkg.com/wireframes@latest/dist/wireframes/wireframes.css" />
