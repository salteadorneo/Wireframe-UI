## Installation

```bash
npm install wireframes@latest
```
```bash
npm install wireframes-react@latest
```
```bash
npm install wireframes-vue@latest
```

## Usage

```html
<wf-image aspect-ratio="16/9" />

<wf-title>
    Hello world!
</wf-title>

<wf-button>
    Click me!
</wf-button>
```
