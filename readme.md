# Wireframes

Web Components UI library for building wireframes and prototypes

## Installation

```bash
npm i wireframes
```

```bash
pnpm i wireframes
```

```bash
yarn add wireframes
```

## Usage

```html
<wf-image aspect-ratio="16/9" />

<wf-title>
    Hello world!
</wf-title>

<wf-button>
    Click me!
</wf-button>
```
