Installation Wireframes is available as an npm package. You can install it in your project using the package manager of your choice. React npm pnpm yarn bun 
npm i wireframes-react
 
pnpm i wireframes-react
 
yarn add wireframes-react
 
bun add wireframes-react
 Vue npm pnpm yarn bun 
npm i wireframes-vue
 
pnpm i wireframes-vue
 
yarn add wireframes-vue
 
bun add wireframes-vue
 JavaScript npm pnpm yarn bun 
npm i wireframes
 
pnpm i wireframes
 
yarn add wireframes
 
bun add wireframes
 After installing, you need to set up at the root of your application. 
import { defineCustomElements } from 'wireframes/loader';
defineCustomElements();
CDN Custom Elements Alternatively, you can use the CDN to load the components in your project. 
<script type="module" src='https://www.unpkg.com/wireframes@latest/dist/wireframes/wireframes.esm.js'></script>
Example 
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script type="module" src='https://www.unpkg.com/wireframes@latest/dist/wireframes/wireframes.esm.js'></script>
</head>

<body>
    <wf-title tag="h1">Login</wf-title>
    <wf-input placeholder="Username"></wf-input>
    <wf-input placeholder="Password" type="password"></wf-input>
    <wf-button>Login</wf-button>
</body>

</html>
CDN Styles You can use the CDN to load the styles into your project. Custom Elements is not included. Normally use headers, input, buttons and these will be styled. 
<link rel="stylesheet" href="https://salteadorneo.github.io/wireframes/cdn/wireframes.css" />
Example 
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="https://salteadorneo.github.io/wireframes/cdn/wireframes.css" />
</head>

<body>
    <h1>Login</h1>
    <input placeholder="Username"></input>
    <input placeholder="Password" type="password"></input>
    <button>Login</button>
</body>

</html>
CDN Styles + Tailwind CSS You can use the CDN to load the styles into your project. Custom Elements is not included. Normally use headers, input, buttons and these will be styled. In addition, you will be able to use the full power of Tailwind CSS for structuring. 
<script src="https://salteadorneo.github.io/wireframes/cdn/wireframes.js"></script>
Example 
<!doctype html>
<html>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://salteadorneo.github.io/wireframes/cdn/wireframes.js"></script>
</head>

<body>
    <main class="container mx-auto p-4">
        <h1 class="text-2xl font-medium">Login</h1>
        <input placeholder="Username"></input>
        <input placeholder="Password" type="password"></input>
        <button>Login</button>
    </main>
</body>

</html>
